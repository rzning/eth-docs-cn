## 1. Setting up a Node project

配置一个 Node 项目。

### 安装 Node

```sh
$ node --version
```

### 创建一个项目

```sh
$ mkdir learn && cd learn

$ npm init -y
```

### 使用 `npx`

```sh
$ npx truffle init
```
